# Modern Climb

Modern ladder and vine climbing mechanics for Minecraft Forge 1.8.9.

## Features

- Hold SPACE to climb ladders automatically
- Vanilla ladder movement preserved
- Proper vanilla-style slide down speed
- SHIFT pauses movement on ladders
- Supports ladders and vines
- Lightweight and client-side

## Installation

1. Install Minecraft Forge 1.8.9
2. Put the mod jar into `.minecraft/mods`
3. Launch the game
